# Wheeltech Autonomous Ground Vehicle

Personal independent project · Built from self-sourced WheelTech parts

![Mecanum wheel platform](Mecanum%20wheel%20platform.jpg)
*Full platform*

## Overview

An omnidirectional autonomous platform fitted with an RPLIDAR A1 and an Orbbec Gemini Pro stereo structured-light camera, driven on Mecanum wheels.

![Basic electronics](Basic%20electronics.jpg)
*Electronics*

## Custom Wheel Design

Beyond the stock wheels, I designed a second wheel set as an experiment: lathe/mill-machined metal hubs paired with 3D-printed TPU tires.

Independently built the full platform and integrated the sensor suite, and independently designed, machined, and assembled the custom wheel set.
