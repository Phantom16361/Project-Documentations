# MTRN3100 Micromouse "Manta"

新南威尔士大学机电工程 Level 3 课程 MTRN3100 Robot Design（机器人设计）小组项目 · 组长

## 项目任务

基于课程提供的电路板，设计一台能有效跑完迷宫的机器人（Micromouse）。

## 我的角色

## 自主研发：柔性一体式悬挂系统

前后全向轮使用了一套自主研发的柔性铰链（flexure）一体式 3D 打印悬挂系统，用单一打印件的弹性形变替代传统多组件悬挂机构，减少装配复杂度与故障点。
