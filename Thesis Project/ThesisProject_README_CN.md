# 毕业设计：医学细胞团抓取与心跳检测直角坐标机器人

新南威尔士大学机电一体化与机器人学荣誉学士毕业设计 · 个人项目

## 项目概览

面向医学研究场景设计的高精度直角坐标（Cartesian）机器人，用于抓取心脏细胞团样本，并同步进行心跳检测。重复定位精度达 **0.01mm**。

![Pick and place machine - View A](Pick%20and%20place%20machine%20VIEW_A.jpg)
*整机视图 A*

![Pick and place machine - View B](Pick%20and%20place%20machine%20VIEW_B.jpg)
*整机视图 B*

机构采用双轴丝杆传动的 XY 平台，配合可更换的多孔培养板（如图中六孔板），实现对指定孔位的精确定位与抓取操作。
