# Thesis: Cartesian Robot for Medical Cell-Aggregate Grasping & Heartbeat Detection

UNSW Mechatronic Engineering & Robotics (Honours) thesis project · Individual project

## Overview

A high-precision Cartesian robot designed for a medical research application — grasping cardiac cell-aggregate samples while simultaneously performing heartbeat detection. Repeatability of **0.01mm**.

![Pick and place machine - View A](Pick%20and%20place%20machine%20VIEW_A.jpg)
*Full view A*

![Pick and place machine - View B](Pick%20and%20place%20machine%20VIEW_B.jpg)
*Full view B*

The mechanism is a lead-screw-driven XY gantry paired with interchangeable multi-well culture plates (e.g. the 6-well plate shown), enabling precise positioning and grasping at a specified well.
